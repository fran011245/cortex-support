# Thoth Demo Video Script (Hackathon / Internal Demo)

**Target length**: 2:30 – 3:00 minutes  
**Tone**: Professional, calm, confident (matching the product)  
**Recording tips**: Use the final `Thoth.app` (not dev build). Record at 1440p or higher. Show the mouse, narrate benefits for real support agents. Use the built-in macOS screen recorder or CleanShot/Loom.

## Script

**[0:00 – 0:15] Hook + Value Prop**

"Bitfinex support agents handle sophisticated traders and corporate accounts every day. Every reply has to be direct, security-aware, and perfectly on-tone — no fluff, no emojis, precise terminology.

Thoth is their always-on, 100% local co-pilot that helps them draft excellent responses in seconds while giving the team complete control over the agent's voice and knowledge."

**[0:15 – 1:00] Core Chat Experience**

"Here's the main chat.

I load a fast local model (or use the one already configured).

A customer says their withdrawal is stuck.

Watch the agent stream a professional reply — it asks for the right details, reminds about security, and stays concise.

One click 'Use as Response' and it's ready to paste into the ticket system."

(Show live typing + sources if RAG is active.)

**[1:00 – 1:45] Quick Action Tools**

"Support agents do a lot of repetitive work.

Grammar & Style tool: paste a messy draft → instantly polished to Bitfinex standards.

Smart Translate: English to Spanish (or other languages) while preserving the exact professional tone.

Response Templates: pick 'Withdrawal Issue', add a couple details from the ticket, and get a complete, on-brand reply ready to customize."

**[1:45 – 2:30] The Killer Feature: Full Customization in Settings**

"But the real power is here — Settings.

The entire personality lives in an editable system prompt — pre-filled with the exact Bitfinex support voice we want.

Tone Rules let you choose Professional, Concise, or Detailed presets and fine-tune with toggles.

Extra instructions: 'Always mention the ticket ID at the top' or 'For corporate clients use last name only.'

General settings control model, temperature, max length, and whether to use RAG.

Knowledge Base: point to your folder of help articles and internal runbooks. Hit Rebuild. Now the agent automatically pulls relevant context and cites the sources.

Export the whole configuration as JSON to share with the team or version control it.

Everything updates instantly for the next generation."

**[2:30 – 2:45] Closing + Benefits**

"Thoth doesn't replace agents — it makes every agent faster, more consistent, and perfectly on-brand.

100% local with Tether's own @qvac/sdk. Your data and knowledge never leave the machine.

Faster replies. Higher quality. Full team control.

That's Thoth — the local co-pilot for Bitfinex Support."

## Suggested B-roll / Screen Recording Order

1. App launch + beautiful dark UI
2. New chat + model status
3. Real customer-style message → streaming reply
4. Copy / Use as Response
5. Sidebar tools demo (grammar, translate, template)
6. Open Settings → edit prompt live → send test message to prove change
7. RAG folder select + Rebuild (show progress)
8. Message that triggers RAG → sources appear
9. Export settings
10. Final shot of the .dmg or app icon

## Music / Style Notes

- Calm, modern tech/finance track (no lyrics)
- Clean cuts, subtle zooms on important UI elements
- Show both the output text and the Settings side-by-side when demonstrating control
- End on the Thoth logo / app icon with tagline

Record a couple takes of the streaming reply — it's impressive live.

This script positions Thoth as a serious, production-grade tool for professional support teams.
